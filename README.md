qtile
=====

A few examples [underwhelming pictures](http://imgur.com/a/6aEvR)

deps
====
- libGL
- libSFML2.0
- jnxlibc (included)

installation
============
- Update the submodule and install it 

```
cd jnxlibc
cmake .
make
sudo make install
```
Build qtile 
```
cmake .
make
./qtile #run
```
